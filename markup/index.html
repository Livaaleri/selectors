Селекторы

Теговый селектор позволяет задавать стили для всех элементов данного типа (тега).

Inline приклад: h1 { color: orange; }

Селектор идентификатора - позволяет задавать стили для конкретного элемента (id должен быть уникальным!). Записывается он почти так же, как и класс, только в CSS вместо точки ставится символ решетки #:

Inline пример: #example { background-color: blanchedalmond;color: olivedrab;}

Селектор класса - позволяет задавать стили для одного и более элементов с одинаковым именем класса, размещенных в разных местах страницы. !!! Вы можете использовать любое имя для вашего класса, но оно не должно начинаться с цифры.

Inline пример: .underline {text-decoration: underline;}

Селектори за вкладеністю

  • Селектор потомка
    1. Позволяет задавать стили к элементам, расположенным внутри элемента-контейнера;
    2. Синтаксис: a b { ... }
    3. Inline пример: ul ol li { margin: 10px; background-color: bisque;}
  • Дочерний селектор
    1. Позволяет задавать стили к элементам, которые являются прямыми потомкам (т.е. идут сразу за родительским элементом и между ними нет других элементов
    2. Синтаксис: a > b { … }
    3. Inline пример: ol>li { margin: 10px; color: rgb(238, 139, 0); }
  • Соседний селектор
    1. Выбирает элемент, который находится непосредственно после указанного элемента, если у них общий родитель.
    2. Синтаксис: a + b { … }
    3. Inline пример: li + li { text-shadow: 1px 1px yellow; text-decoration: underline wavy; }
  • Родственный селектор
    1. Выбирает все элементы, которые находятся после указанного элемента, если у них общий родитель.
    2. Синтаксис: a ~ b { … }
    3. Inline пример: main ~ footer { background-color: blanchedalmond;}

Cелектор атрибута - позволяет задавать стили к элементам на основе имени атрибута или значения атрибута;

Синтаксис: a[attr=”value”] { … }

Inline пример: [href^="http"] { color: purple; font-weight: 600; }
